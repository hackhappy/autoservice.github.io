1. File Structure

HTML files

index-{color}.html - contains the full HTML code of the template with color skin preset.

CSS Files

/css folder contains:

  • /sass folder with SASS sources
    • /bourbon folder - Bourbon SCSS/SASS mixin library
    • /skins folder - skins variations with SASS color variables
    • main.sass - main styles theme SASS sources
  • /skins folder contains compiled skins styles
  • main.css - compiled /sass/main/sass file
  • media.css - media breakpoints styles

Libraries

/libs folder contains all Front-End plugins with CSS & JS files.

  • /animate - contains animate.css plugin
  • /bootstrap - contains bootstrap grid 3.3.2 version
  • /html5shiv, /respond - HTML5 & responsive support for IE
  • /jquery - fast, small, and feature-rich JavaScript library
  • /magnific - Magnific Popup is a responsive lightbox & dialog script with focus on performance and providing best experience for user with any device (for jQuery or Zepto.js).
  • /modernizr - Modernizr is a JavaScript library that detects HTML5 and CSS3 features in the userâ€™s browser
  • /myFonts - Custom thematic icons font

    Corresponding symbols (Example use: <i class="my my_a"></i>):

  • /owl - Owl Carousel 2: Touch enabled jQuery plugin that lets you create a beautiful responsive carousel slider.
  • /plugins-scroll - Chrome Smooth Scroll plugin
  • /retina - retina.js is an open source script that makes it easy to serve high-resolution images to devices with retina displays
  • /stellar - stellar.js - parallax plugin
  • /waypoints - Waypoints - is the easiest way to trigger a function for scroll to an element.

mail.php

E-mail send script

prepros.cfg

Prepros configuration file for SASS compilation

2. Installation

Rename any index-{color}.html file to index.html

  1. Start your FTP client (FileZilla) and login to your server
  2. Locate, on your server, the domain folder you wish to upload your template files to. You are going to click and drag every file and folder that you have inside the HTML folder into your FTP client
  3. Once the files are done uploading, you can access your website via browser

3. Customizations

The only things you need to change the template are a text editor and a browser, to see the outcome. For the text editor I recommend brackets, which is free.

Type pagetitle and more meta information

Open index-{color}.html and put you data

<!--  Meta Info -->
<title>ProCourse - Responsive Education Agency Landing Page Template</title>
<meta content="English Courses" name="description" />

Logo

Type you logo text or load logo image here: 
<div class="logo">
	ProCourse <!-- Logo Text -->
</div>

Site name and Description

Type you Site name and Description here: 
<h1>English Courses</h1> <!-- Site Name -->
<h2>Pro Courses of English Language</h2> <!-- Site Description -->

Change phone and city tabs

  1. Change top and footer phones or their count here (search "Top Phones Tabs Start" and Footer Phones Tabs Start" HTML comments):

    <!-- Header phones -->
<div class="tabs">
	<span class="tab" data-tabs="top_tabs">Moscow</span> /
	<span class="tab" data-tabs="top_tabs">St. Petersburg</span>
</div>
<div class="tab_content">
	<div class="tab" data-tabs="top_tabs">+7 (999) 999-99-99</div> /
	<div class="tab" data-tabs="top_tabs">+7 (000) 000-00-00</div>
</div>

<!-- Footer phones -->
<div class="tabs">
	<span class="tab" data-tabs="bottom_tabs">Moscow</span> /
	<span class="tab" data-tabs="bottom_tabs">St. Petersburg</span>
</div>
<div class="tab_content">
	<div class="tab" data-tabs="bottom_tabs">+7 (999) 999-99-99</div> /
	<div class="tab" data-tabs="bottom_tabs">+7 (000) 000-00-00</div>
</div>

    Each phone line corresponds to one city of the corresponding line.

  2. Change city forms (find "option" tags and change content):

    <select name="city">
	<option>Moscow</option>
	<option>St. Petersburg</option>
</select>

  3. Change address of city here:

    <!-- Contacts Tab Address Item Start --> (First City)
<div class="tab_item" data-tabs="map_tabs">
	Big Ovchinnikovsky per., 24, p. 4, Moscow
</div>
<!-- Contacts Tab Item End -->

<!-- Contacts Tab Address Item Start --> (Second City)
<div class="tab_item" data-tabs="map_tabs">
	Nevsky Pr., 134, St. Petersburg, Russia, 191036
</div>
<!-- Contacts Tab Item End -->

  4. Change Google Maps

    1. Paste you Google Maps API key here:

      <!-- Google Map API -->
<script type="text/javascript" src="https://maps.googleapis.com/maps/api/js?key=YOU-API-KEY&sensor=false"></script>

    2. Settings Contacts Map coords and zoom for all cities (find Contacts Tab HTML comments)

      <span class="tab button btn_white"
	class="tab button btn_white"
	data-map-coord-first="55.749792"
	data-map-coord-second="37.632495"
	data-zoom="12"
	data-tabs="map_tabs">Moscow</span> <!-- Contacts Tab -->

<span class="tab button btn_white"
	class="tab button btn_white"
	data-map-coord-first="59.9174455"
	data-map-coord-second="59.9174455"
	data-zoom="12"
	data-tabs="map_tabs">St. Petersburg</span> <!-- Contacts Tab -->

Change content

  1. Setting Sales tabs here (find "Top Tabs Start HTML" comment):

    <span class="tab" data-tabs="sales_tabs">Sales</span>

    and change or add an equivalent container:

    <div class="tab_item" data-tabs="sales_tabs">
...

  2. Change the list of advantages here (find "Professional Item Start"):

    <div class="profi_right col-md-10 col-sm-10">
	<h3>Experience</h3>
	<p>We have been successfully working in the market of educational services for <strong>over 10 years</strong>, as evidenced grateful reviewed our numerous customers.</p>
</div>

    You can replace "my my_g" class with Font Awesome class Icon. List of FA icons classes.

    <div class="prifi_left col-md-2 col-sm-2">
	<i class="my my_g"></i>
</div>

  3. Change Students feedback section: find "Review Item Start" HTML comment and change or add new reply
